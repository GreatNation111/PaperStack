# PaperStack Roadmap

This is a living document that outlines planned improvements and features for PaperStack. Items are listed roughly in order of priority, but this may change based on community feedback and contributions.

## Short-Term

- [ ] Improve course search with better filtering by department, level, and semester
- [ ] Improve PDF preview experience (faster loading, better mobile rendering)
- [ ] Add better validation for admin document uploads
- [ ] Improve mobile PWA install experience and onboarding
- [ ] Add accessibility improvements (keyboard navigation, screen reader support, ARIA labels)

## Medium-Term

- [ ] Add contributor recognition page to highlight active contributors
- [ ] Improve documentation for Firebase setup and self-hosting
- [ ] Add support for multiple universities and institutions
- [ ] Implement course bookmarking and study lists
- [ ] Add exam countdown and reminder notifications

## Long-Term

- [ ] Lecturer repeat question analysis and insights
- [ ] Step-by-step detailed solutions for past questions
- [ ] Offline mode improvements for the PWA
- [ ] Internationalization (i18n) support
- [ ] API for third-party integrations
